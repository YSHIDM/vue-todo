# vue-todo

## 简介

一个简单的待办管理小工具

使用 vue2 + vant

## 功能

1. 待办增删改查
2. 待办归档
3. 待办回收站
4. 待办倒计时页面版
5. 设置待办节点(待完善)

## 命令

```shell
# 下载代码
git clone https://github.com/YSHIDM/vue-todo.git
# 安装 npm 包
npm install
# 启动本地服务
npm run serve
# 编译
npm run build
# 检查及修改代码
npm run lint
```
